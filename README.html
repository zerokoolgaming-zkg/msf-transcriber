Setup – ZKG Counter Upload

  1. Google Sheet: use the Sheet ID 1Mq88NZUs6rIsbQFGmR_4koqxZofYeFS063-S81GtShk and ensure a tab named Counters with columns: 
    A:Team Member 1 ... E:Team Member 5, F:Counter Character 1 ... J:Counter Character 5, K:Season, L:ROOM, M:PUNCHUP OR DOWN, N:TCP TEAM, O:TCP COUNTER, P:TCP Difference, Q:Victory Points, S:(Notes)
  2. Apps Script Backend: create a new Google Apps Script attached to the Sheet and paste the provided Code.gs. Deploy as Web app with Anyone with the link (or your preferred auth). Copy the deployment URL.
  3. Frontend Config: open script.js and set CONFIG.backendUrl to that Apps Script URL. Commit/push to GitHub Pages.
  4. (Optional) Portrait matching:
    • Place character portraits in /portraits (PNG/JPG). Use character names as file names.
    • Open tools/hash-builder.html in your browser, click Build Hashes, then Download JSON. Save as portraits_hash.json in the project root.

On the site, click Upload Screenshot. While processing, you’ll see a subtle glowing Uploading message. When done, it changes to Completed, waits 5 seconds, then auto-refreshes for the next upload.
