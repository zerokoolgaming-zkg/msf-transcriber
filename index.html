How this works

  1. Click Upload Screenshot. You can select one or more images.
  2. We OCR the screenshot to pull totals (Attack vs Defense), VP, etc.
  3. We compute Punchup / Punchdown and TCP Diff.
  4. We POST a new row into the app (tab Counters).

Heads-up: automatic character-name detection from portraits is best-effort. For high accuracy, build a portrait hash via the Hash Builder and include it as portraits_hash.json.
